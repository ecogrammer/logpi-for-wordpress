== Installation ==

* A plug-in installation screen is displayed on the WordPress admin panel.
* It installs it in `wp-content/plugins`.
* The plug-in is made effective.
* It registers on the widget management screen.
* The name of the Logpi user to be displayed is set.
 * タイトル:title
 * ユーザー名:user account of Logpi.
 * 表示件数:default 5, max to 10.
 * 投稿日時を表示:timestamp.
 * ログリンク:example '>>' or 'detail'. When inputting as 'all', It becomes effective for the message the link.
 * 記事内リンク:It links effectively in the article. 
 * ユーザーリンク @リプライ:The reply logs are effectively linked.
 * UTF8 Encode:UTF8 Encode is made effective.
 * 表示ログ:The kind of the displayed log is input.

* Wordpress管理画面にて、プラグインのインストール画面を表示します。
* ZIP形式のプラグインを展開し、wp-content/plugins にインストールします。
* プラグインを有効にする事で利用可能となります。
* ウィジット管理画面を表示して、Logpi#1をサイドバーにドラッグ・ドロップします。
* 表示したいユーザー名、件数を設定します。
 * タイトル:任意のタイトル
 * ユーザー名:ログピのユーザーアカウント
 * 表示件数:未設定時は5件。最大10件
 * 投稿日時を表示:タイムスタンプ表示の有無。
 * ログリンク:ログリンクの表示。例えば '>>' や '詳細' などを設定します。'all'を入力すると全投稿メッセージにリンクが有効になります。
 * 記事内リンク:記事内のリンクを有効にする。
 * ユーザーリンク @リプライ:返信ログリンクを有効にする。
 * UTF8 Encode:メッセージに対してUTF-8エンコードを行います。
 * 表示ログ:表示ログの設定を行います。'you'を入力するとあなたのログ。'follow'で友達のログ。'reply'で返信されたログ。'fav'でLoveしたログを表示します。
